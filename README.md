# e-plantShopping

## Paradise Nursery Shopping Application

A React + Redux shopping cart application for browsing and purchasing houseplants online.

## Features

- Landing page with company info and Get Started button
- Product listing page with 18 houseplants across 3 categories
- Add to Cart functionality with disabled state after adding
- Shopping cart with quantity controls, item totals, and delete
- Dynamic cart icon showing total item count
- Built with React, Redux Toolkit, and Vite
